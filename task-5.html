TASK 5 - My favorite project

I have worked on numerous projects where I implemented cool GSAP animations. If I had to pick my favorite, it would be the Vegeta.com website.

For this project, I was responsible for most of the animations on the site, but I'm particularly proud of the 'Vegeta History' page, which I created entirely by myself.

My favorite feature on this page is the slider that I developed using the Swiper.js library, seamlessly integrated with GSAP to create a smooth and engaging user experience.
